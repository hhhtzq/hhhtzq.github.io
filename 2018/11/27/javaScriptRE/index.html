JavaScript Regular Expression

Posted on 2018-11-27 | In JavaScript , Note |
Words count in article: 663Words | Reading time ≈ 2Mins

定义

很多语言中都有用到它,我的理解是它是用来规则表达式的.

应用场景:表单验证,解析url参数,筛选等等….

创建方法

字面量形式

var reg = /a/; ⚠️ a不能放入变量|函数

构造函数形式

var reg = new Reg("a"); ⚠️a可以放入变量或函数

RE的方法

.text .exec
看内容是否与正则匹配 看内容是否与正则匹配
返回:false或true 匹配返回Array,不匹配返回null
暂时未知 不支持全局g

Metacharacter

元字符 描述 实例
[ ] 匹配 [ ] 中任意一个字符
[^] 匹配除了 [ ]中的字符
[0-9] 匹配0-9的任意数字,数字可以改变 var reg = /[0-9]/;var num = 100;返回true ,是因为先看到了1
[a-z] 匹配a-z的任意小写字母
[A-Z] 匹配A-Z的任意大写字母 [0-9a-zA-Z]可以这样拼接
\d 匹配数字
\D 匹配非数字
\w 匹配数字,字母和下划线 =[0-9a-zA-Z_]
\W 取返
\s 匹配空格
\S 匹配非空格
\b 匹配边界,我对边界的理解为空格 var reg = /\ba/;var str = " a" 返回为true
. 匹配除了换行符\n以外的都返回true
\ 转义符,\.只匹配 .

限定符

限定符 描述
^ 开始
$ 结束

量词

量词 描述
{n,m} 最少n次最多m次
* 0-∞
+ 1-∞
? 0-1

选择符

/a|b/选择匹配a或者b

修饰符

叙事符 描述 实例
i 不区分大小写 /[A-Z]/i == new Reg("[A-Z]","i")
g 全局匹配 var Reg = /a/g ; var str = "abc aa a" ; var str1 = str.replace(Reg,"#")返回结果为#bc ## #
m 多行匹配,不怎么用到 可以组合使用/a/igm

分组

() 比如 /^(abc)$/ 表示以 abc 开始 并且以 abc 结束,可以看成是一个组合.

字符类

RegExp.$1....$99 必须 先使用分组 再运行正则 ,返回的是匹配的内容.

字符串方法

字符串正则方法 描述
search 返回位置,如果不匹配返回-1,不支持全局 g ,只能找到第一个
match 匹配返回数组,不匹配返回null,支持全局 g
replace 替换功能,支持全局 g
